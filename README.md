# Evia Ultimate Hotel Booking – Ethiopia

A comprehensive, production‑ready hotel booking platform tailored for the Ethiopian market, featuring:

- 🇪🇹 **Local payment integration**: Telebirr, CBE Birr, Chapa (with mock adapters, easily replaceable)
- 🗣️ **Bilingual interface**: Amharic & English (with `next-i18next`)
- 🏨 **Full hotel & room management** (admin dashboard)
- 🚖 **Extensions ready**: AI assistant, blockchain escrow, freelance marketplace (folders in `/extensions`)
- 🧪 **Complete test suite**: Unit, integration, and e2e tests
- 🐳 **Dockerized**: PostgreSQL, Redis – easy local setup
